# Module 1: Enterprise Knowledge Workspace

中文项目名：企业知识工作台：面向知识复用与决策准备的 Agent 工作流演进

## 用途

该目录承载模块 1 的中文作品集项目页、支撑文档、图表和脱敏截图。

当前公开阅读入口：

- [中文项目页](index.html)
- [中文产品设计文档 PDF](../../assets/docs/enterprise_knowledge_workspace_product_design_cn.pdf)
- [English Product Design Brief](../../assets/docs/enterprise_knowledge_workspace_product_design_brief_en.pdf)

## 内容边界

- 核心对象：知识资产、历史经验、决策依据。
- 核心问题：企业过去积累的资料、经验和判断，如何被检索、引用、复盘和沉淀。
- 页面叙事重点：可信上下文层、Agent 工作流、人机确认、badcase / eval、版本迭代。
- 不展开完整任务状态、责任人、优先级、周报和推进闭环；这些内容留给模块 2。

## 目录结构

- `index.html`：中文作品集项目页。
- `assets/diagrams/`：模块 1 正式图表。
- `assets/screenshots/`：脱敏页面截图和产品界面截图。
- `content/`：文案草稿、结构草案、临时内容片段。
- `examples/`：脱敏样例配置与样例知识库。
- `enterprise_knowledge_workspace_case_study_en.md`：英文案例草稿。
- `企业知识工作台_RAG到Agent工作流_产品设计文档.md`：历史支撑文档。

## 页面口径

页面不要写成 RAG demo 或本地知识库工具。当前统一表达为：

> 企业知识工作台：面向知识复用与决策准备的 Agent 工作流演进。

RAG 只作为可信上下文层的一部分，用于承接资料范围、引用溯源、权限边界、知识更新、人工核验和评估迭代。
