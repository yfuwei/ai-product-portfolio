# 截图说明

本目录保存本地 RAG 作品集案例使用的脱敏截图。

当前图片由脱敏截图夹具生成，使用 fake `sample-vault` 内容，不包含真实 Obsidian 笔记名、真实 OneDrive 路径、Chroma 数据或私人原始文档。

## 截图清单

| 文件 | 用途 | 脱敏规则 |
|---|---|---|
| `notion-style-gui.png` | 展示 Notion 风格 GUI 总览 | 使用假问题和假来源名称 |
| `search-workspace.png` | 展示检索输入、结果列表和片段详情 | 使用 `sample-vault/...` 示例路径 |
| `answer-with-citations.png` | 展示带引用的回答模式 | 只使用样例笔记 |
| `indexing-workspace.png` | 展示小批量索引控制和日志预览 | 使用示例配置路径 |

## 不应包含

- 真实 Obsidian 笔记标题。
- 真实 OneDrive 文件路径。
- 个人文档内容。
- Chroma 数据库截图。
- 带本机绝对路径的终端日志。

## 重新生成说明

这些截图不是直接从私人索引截取，而是从临时脱敏 HTML 夹具生成。这样既能保留产品界面和工作流，又适合公开发布。
