# B端审批协同平台流程说明

## 核心流程

```text
草稿 -> 已提交 -> 预审中 -> 待评审 -> 评审完成 -> 已归档
              └-> 退回补充 -> 已提交
```

## 角色权限

| 角色 | 可执行动作 |
| --- | --- |
| 提交方 | 新建报审、上传材料、补充材料、查看状态 |
| 预审方 | 检查材料、退回补充、通过预审 |
| 评审方 | 查看材料、记录意见、确认结论 |
| 管理者 | 查看流程状态、追踪逾期事项、导出记录 |

## 结果表达

该流程将原本依赖线下沟通和人工追踪的报审流程转为可提交、可预审、可追踪、可沉淀的协同流程。
