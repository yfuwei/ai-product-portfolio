# AIGC视觉内容生成工作流 PRD 草稿

## 1. 项目背景

前期方案表达中经常需要快速形成建筑概念图、效果意向图、空间氛围图和汇报辅助配图。传统出图流程较重，需求口径抽象，多轮沟通和修改成本高。

## 2. 产品目标

将模糊出图需求拆解为结构化输入，通过 Prompt 模板、风格标签和多版本生成流程，辅助前期方向探索和汇报素材准备。

## 3. 目标用户

- 产品/方案负责人
- 设计协同人员
- 汇报材料准备人员

## 4. 核心流程

1. 选择项目类型
2. 选择建筑风格
3. 选择使用场景
4. 选择画面视角
5. 填写表达重点
6. 生成结构化 Prompt
7. 输出多版本结果
8. 人工筛选和后修
9. 标记适用场景与边界

## 5. Demo 范围

当前 Demo 使用预设案例模拟 AIGC 工作流，不接入真实 Stable Diffusion 或 ComfyUI 服务。

## 6. 验证口径

- 15 个典型出图场景
- 评估维度：需求贴合度、风格稳定性、表达可用性、人工修改成本
- 结果：13 个场景达到“基本可用及以上”

## 7. 边界判断

适合：概念方向、空间氛围、汇报配图、封面背景。

不适合：正式效果图、逻辑型分析图、精确总平关系、施工或报规用途。
