# Module 2: AI Business Requirement Flow Workspace

中文项目名：AI 业务需求流转工作台：从多源业务信号到需求分流与任务闭环

## 用途

该目录用于承载模块 2 的新版中文项目页、图片资产和内容草稿。

模块 2 是新增核心项目，不应沿用旧会议纪要、需求卡片或文档处理目录。

## 内容边界

- 核心对象：当前需求、任务、责任链路、状态流转。
- 核心问题：新需求从会议、CRM、销售、客服、售后、质量反馈和项目沟通中产生后，如何被识别、确认、拆解、分流、执行和复盘。
- 不写成会议纪要工具、文档总结工具或模块 1 的知识库变体。

## 目录结构

- `assets/diagrams/`：正式图表。
- `assets/screenshots/`：页面截图、产品界面截图。
- `content/`：文案草稿、结构草案、临时内容片段。
