# B端审批协同平台工作流优化升级

这是 GitHub 作品集中的 B端流程协同项目资料夹。

## 文件

- `index.html`：项目案例页。
- `assets/workflow.svg`：升级后主流程图。
- `assets/modules.svg`：产品模块框架图。
- `assets/status.svg`：状态流转图。
- `diagrams/workflow.mmd`：主流程 Mermaid 源文件。
- `diagrams/modules.mmd`：模块框架 Mermaid 源文件。

## 口径

本项目是真实 B 端平台升级项目，不是 AI 项目。页面表达重点是复杂流程拆解、角色权限、字段状态、项目推进和上线落地能力。
