# Module 4: AI Evaluation and Governance Framework

中文项目名：AI 应用评估与治理框架：从 Demo 到可上线、可迭代、可复制

## 用途

该目录用于承载模块 4 的新版中文专题页、图片资产和内容草稿。

模块 4 是作品集的高站位收束，不是普通项目复盘或泛方法论总结。

## 内容边界

- 核心对象：AI 场景机会评估、badcase、人审、权限、版本管理、回归测试、上线准入和复制机制。
- 核心问题：哪些 AI 场景值得做、是否可控、是否可上线、是否能从试点复制到其他业务场景。
- 不写成空泛治理口号，也不包装成已经真实上线的企业治理平台。

## 目录结构

- `assets/diagrams/`：正式图表。
- `assets/screenshots/`：页面截图、产品界面截图。
- `content/`：文案草稿、结构草案、临时内容片段。
