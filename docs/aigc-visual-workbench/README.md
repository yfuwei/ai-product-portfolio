# 建筑方案视觉生成工作台（AIGC）

这是 GitHub 作品集中的 AIGC 子项目页面资料夹。

## 文件

- `index.html`：项目案例页。
- `assets/hero-workbench.svg`：封面工作台示意图。
- `assets/architecture.svg`：产品框架图。
- `assets/user-flow.svg`：用户流程图。
- `assets/workbench-overview.svg`：页面原型总览图。
- `assets/evaluation-chart.svg`：验证结果图。
- `diagrams/architecture.mmd`：产品框架 Mermaid 源文件。
- `diagrams/user-flow.mmd`：用户流程 Mermaid 源文件。

## 口径

本项目是自主产品原型和场景验证，不是商业化上线产品。页面表达重点是产品思考、工作流设计、试用反馈和验证边界。
