# Module 3: AI Content Operations Workflow

中文项目名：AI 内容运营工作流：从营销 Brief 到多渠道内容协同与反馈沉淀

## 用途

该目录用于承载模块 3 的新版中文项目页、图片资产和内容草稿。

后续不再把模块 3 写成文生图、AIGC 设计支持或素材生成执行项目。

## 内容边界

- 核心对象：内容运营流程、营销 brief、品牌规则、多渠道适配、审核反馈、投放反馈和内容资产沉淀。
- 核心问题：如何把内容生产从零散需求响应，转化为可复用、可审核、可反馈的 AI 内容运营工作流。
- Stable Diffusion、ComfyUI、Prompt 等只作为局部工具，不作为主叙事。

## 目录结构

- `assets/diagrams/`：正式图表。
- `assets/screenshots/`：页面截图、产品界面截图。
- `content/`：文案草稿、结构草案、临时内容片段。
- `index.html`：模块 3 可编辑页面原型入口，包含 Brief 结构化、创意方向选择、多渠道版本管理、品牌审核、效果复盘和作品集展示说明。
